# 标准节点 - 计算任务 接入细则 (费用月结)

### ● A任务随时可以上，当月满15天计费，不满15天下月计费。   
### ● F任务和G任务，需要等待业务方通知，请留意群内公告。
### ● 结算方式为每月结算，可以对公结算。
<br>

## A任务 - 公网动态IP或内网IP汇聚 (仅支持实体机直接安装)
### 接入带宽要求
**● 单线路上行带宽不小于 100 Mbps**  
**● 汇聚后上行带宽不小于 3.0 Gbps**  
**● 线路为中国大陆境内三大运营商线路 (中国电信，中国联通，中国移动)**  
**● 电信或联通：每条线路均具有公网动态IP**  
**● 移动：每条线路均具有公网动态IP或内网IP支持全锥型 (FULL CONE, NAT1)**  
### -> [接入教程](https://shimo.im/docs/QqJtpkCkYvRTQWvx/)
<br>
